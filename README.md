# Time manager API
This app is an api to hellp you orgenaize your long awaiter Trip!

### INSTALL
---
run `npm install`


### USAGE
---
run `node index.js`
default port is `3000`


### Endpoints
---
* `GET` /selectDates
* `GET` /selectDates/:id
* `POST` /selectDates
* `PUT` /selectDates
* `DELETE` /selectDates/:id
